Base Image
==========

This is base image, contain common package for builing and version control like:
cvs, bazaar, git, subversion

Avaliable distro:
 - CentOS 5
 - CentOS 6
 - CentOS 7
 - Debian Wheezy
 - Debian Jessie
 - Debian Stretch
 - Ubuntu Precise
 - Ubuntu Trusty
 - Ubuntu Xenial
